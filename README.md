# Restaurants Performance Analysis Dashboard

## 📌 Project Overview
This project features a comprehensive **Power BI Dashboard** designed to analyze restaurant performance, sales trends, and customer preferences. The interactive dashboard translates raw restaurant industry data into actionable business insights, helping stakeholders monitor key performance indicators (KPIs), track revenue growth, and identify top-performing culinary trends and brands.

---

## 📊 Key Insights & Dashboard Metrics
Based on the visual data analyzed in the dashboard (**image_76c32b.jpg**):

*   **Total High-Level KPIs:**
    *   **Total Orders:** 150K
    *   **Total Revenue:** 987M
    *   **Total Quantity Sold:** 2M
    *   **Total Restaurants:** 149K
*   **Sales by Cuisine:** 
    *   North Indian cuisines dominate the sales distribution capturing the highest shares (26% and 24%), followed by Chinese and Indian fusion variations.
*   **Top Restaurants Performance:**
    *   **Domino's Pizza** leads significantly in both sales ($5.0M) and total order quantity ($8.4K$).
    *   Other key players include **Pizza Hut**, **Sweet Truth**, **Kouzina Kafe**, and **Biryani House**.
*   **Revenue Trends (Years Wise):**
    *   The business experienced a massive spike in revenue in **2018**, peaking at **6.6M**, followed by a decline and stabilization around **2.4M** by 2020.
*   **Sales by Rating:**
    *   A massive majority of the revenue ($538M) is driven by highly-rated items/restaurants (specifically around the 3.89 rating mark).

---

## 🛠️ Features & Functionalities

*   **Interactive Filters (Slicers):** Dynamic drill-down capability by **Gender**, **Year**, **Month**, **Quarter**, and **Family Size** to customize views instantly.
*   **Demographic Segmentation:** Includes demographic filters like **Veg or Non-Veg** preferences, **Educational** background, and **Marital Status** for granular customer profiling.
*   **One-Click Reset:** An **"All Clean"** button implemented using bookmarks to clear all filters seamlessly.
*   **Advanced Visualizations:** A mix of KPI cards, Donut charts, Bar charts, Pie charts, and Line graphs for optimized storytelling.

---

## 💻 Tech Stack Used
*   **Tool:** Power BI Desktop
*   **Data Transformation:** Power Query (Data Cleaning & ETL)
*   **Modeling:** DAX (Data Analysis Expressions) for custom metrics
*   **Design:** Modern UI layout with clean section dividers and cohesive color palette
